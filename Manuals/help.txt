NAME
    help - display command help information

SYNOPSIS
    help
    help <command>
    help --help

DESCRIPTION
    Displays a list of available commands or detailed help
    information for a specific command.

OPTIONS
    --help
        Displays help information about the help command.

EXAMPLES
    help
        Lists all commands.

    help ls
        Shows help information about the ls command.

NOTES
    help displays command metadata only. For full documentation,
    use the man command.
